##how to demo

1. checkout
1. run `npm install`
1. run `spm install`
1. run `grunt`
1. view files in `/demo`

##how to use

1. run `spm install pandora/overlay`
1. write `require('pandora/overlay/VERSION.NUMBER/overlay')`
